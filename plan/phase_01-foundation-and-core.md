| `.to_sql()` | Generate (sql, params) tuple |

### Model Trait (`src/model/model.rs`)

Generated by `#[derive(Model)]`. Methods:

- `table_name() -> &'static str`
- `primary_key() -> &'static str`
- `columns() -> &'static [&'static str]`
- `soft_delete_column() -> Option<&'static str>`
- `timestamps_enabled() -> bool`
- `query() -> QueryBuilder<Self>`
- `find(id) -> QueryBuilder<Self>`

### `#[model(...)]` Macro Attributes

| Attribute | Effect |
|-----------|--------|
| `table = "name"` | Override table name |
| `primary_key = "col"` | Override PK column |
| `#[model(primary_key)]` on field | Mark field as PK |
| `soft_delete` | Enable soft-delete filtering |
| `timestamps` | Enable auto `created_at` / `updated_at` injection |
| `#[model(skip)]` on field | Exclude from columns list |
| `#[model(column = "name")]` on field | Map to different DB column |
| `belongs_to_many(...)` | Many-to-many (basic) |

### PgModel Trait (`src/model/pg_model.rs`)

All methods are async and return `Result<_, sqlx::Error>`:

| Method | Description |
|--------|-------------|
| `all(pool)` | `SELECT *` |
| `find_by_pk(pool, id)` | `SELECT * WHERE pk = id` ‚Üí `Option<Self>` |
| `find_or_404(pool, id)` | Like above but errors if None |
| `first(pool)` | First record |
| `first_or_404(pool)` | First or error |
| `find_where(pool, builder)` | Custom query |
| `count(pool)` | `COUNT(*)` |
| `count_where(pool, builder)` | Conditional count |
| `create(pool, data)` | INSERT, returns rows affected |
| `create_returning(pool, data)` | INSERT ‚Ä¶ RETURNING * |
| `update_by_pk(pool, id, data)` | UPDATE by PK |
| `update_where(pool, builder, data)` | Conditional UPDATE |
| `delete_by_pk(pool, id)` | DELETE by PK |
| `delete_where(pool, builder)` | Conditional DELETE |
| `bulk_create(pool, rows)` | Multi-row INSERT |
| `restore(pool, id)` | Clear `deleted_at` |
| `force_delete(pool, id)` | Hard DELETE (soft-delete models) |
| `sum / avg / min / max` | Aggregates |
| `exists / exists_where` | Existence check |
| `pluck / pluck_where` | Single column retrieval |
| `update_all / update_all_where` | Mass update |
| `delete_in(pool, col, ids)` | DELETE WHERE col IN (...) |
| `upsert / upsert_returning` | ON CONFLICT upsert |
| `paginate(pool, page, per_page)` | Returns `Page<Self>` |

Same API exists on `SqliteModel` and `MyModel` (MySQL).

### Relations (`src/relations/relations.rs`)

| Type | Description |
|------|-------------|
| `HasMany<P, C>` | One parent ‚Üí many children |
| `HasOne<P, C>` | One parent ‚Üí one child |
| `BelongsTo<P, C>` | Child ‚Üí parent |
| `lazy::load_has_many` | Batch load children (PG) |
| `lazy::load_has_one` | Load single child (PG) |
| `lazy::load_belongs_to` | Load parent (PG) |

Eager loading via `.with("relation_name")` on QueryBuilder.
